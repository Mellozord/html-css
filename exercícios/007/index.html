Outras formatações

Monoespaçado

Para escrever códigos a fonte monoespaçada é ideal, mas continua não respeitando indentação

                Para indentar
                é necessário colocar
                o texto dentro
                da tag <pre>

Citações simples e completas

Para escrever frases com citações simples as aspas podem ser adicionadas com a tag <q>

Para escrever frases com citações completas a formatação pode ser adicionada com a tag <blockquote>

e para ser identificada no mecanismo de pesquisa podemos inserir o link de citação no código com a tag <cite>

Abreviações

Para escrever frases com abreviações, como HTML, usamos a tag <abbr>

Texto invertido

Para textos invertidos usamos a tag <bdr> e os sentidos rtl e ltr Não sei porque alguém usaria isso
